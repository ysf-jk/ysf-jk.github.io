～巫女の迷路～　説明書的なもの

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
☆内容
迷路に迷い込んだ巫女さんをゴールまで導いてあげましょう

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
☆操作方法
↑↓→←キーで巫女さんが移動します
Rキーでタイトルに戻ります

----------------------------------------------------------------------------------------------------------------------------------------------------------------------------
☆フィールドについて
このゲームには５種類のブロックが出てきます

・スタート/ゴールブロック
スタートとゴールの位置を分かりやすくするためのブロック
スタート：青　ゴール：赤

・床ブロック
ただの床
色は白

・壁ブロック
ただの壁、通ることは不可能
色は灰色

・魔法陣
ワープすることが可能
色はいろいろ

・黒巫女さん
黒い巫女さん
スタートまで戻してくれる
見た目はプレイヤーの色違い
